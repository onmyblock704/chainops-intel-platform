# ChainOps Intelligence Terminal

Realtime blockchain intelligence, smart contract risk analysis, and cross-chain behavioral monitoring platform built with Next.js, TypeScript, Node.js, WebSockets, and Ethereum infrastructure APIs.

ChainOps is designed as an institutional-style blockchain operations terminal that combines live on-chain telemetry, contract forensics, smart money analytics, and streaming intelligence systems into a unified operational interface.

The platform explores how modern blockchain infrastructure, realtime analytics, and security intelligence can converge into next-generation Web3 monitoring systems.

# Local Development

## Clone Repository


git clone https://github.com/YOUR_USERNAME/chainops-intelligence-terminal.git
cd chainops-intelligence-terminal


---

# Install Dependencies

## Frontend


npm install


## Backend


cd server
npm install

---

# Environment Variables

Create a `.env` file:


DATABASE_URL=

VITE_ALCHEMY_API_KEY=

ALCHEMY_WS_URL=

RPC_ENDPOINT=

VITE_WS_URL=ws://localhost:3000/ws

VITE_WALLETCONNECT_PROJECT_ID=


---

# Run Frontend


npm run dev


# Run Backend


cd server
npm run dev
